<h2> <img src = "https://media2.giphy.com/media/QssGEmpkyEOhBCb7e1/giphy.gif?cid=ecf05e47a0n3gi1bfqntqmob8g9aid1oyj2wr3ds3mg700bl&rid=giphy.gif" width = 32px> Description</h2>
Charades.io is an online implementation of the popular multiplayer game Dumb Charades. The players are divided into two teams and are connected through a video call. Various features have been implemented to allow the players to play the game with ease.
  <h2> <img src = "https://media2.giphy.com/media/QssGEmpkyEOhBCb7e1/giphy.gif?cid=ecf05e47a0n3gi1bfqntqmob8g9aid1oyj2wr3ds3mg700bl&rid=giphy.gif" width = 32px>  Technologies used: </h2>
-ReactJS for frontend<br>
-NodeJS with Express for backend<br>
-Web Sockets for client to client communication<br>
-Agora.io SDK for video calling
